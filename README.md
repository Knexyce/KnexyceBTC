# KnexyceBTC

KnexyceBTC is a script CLI that serves as a Bitcoin manager and E2E encrypted chat platform.

## Pros
1. Unlimited Wallets.
2. Wide support for private keys during transactions.
3. AES-256 encrypted Wallets.
3. A secure E2E encrypted chat platform.
4. Choosable transaction fees.
5. Check any wallets' balance.
6. Auto handles dependencies.
7. A user-friendly CLI.

## Cons
1. No taxing system.
2. Zero external oversight.
3. Anyone's private key can be used for KnexyceBTC.
4. The chat system has no way to be supervised.
5. All Wallets made by KnexyceBTC are directly handled by you only.
6. No record, history, or logs on how you use KnexyceBTC.
7. Highly difficult to trace.
